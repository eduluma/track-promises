# Repo-local LLM notes

Use this directory for repository-specific AI scratch files that should stay local to your machine.

Suggested structure:

- `.llm/memories/` for copied or exported memory notes
- `.llm/prompts/` for reusable prompt drafts
- `.llm/runs/` for local investigation logs

The Copilot `/memories` store is managed separately and does not automatically read from or write to this folder.
If you want notes from `/memories` mirrored here, copy them manually or script that export locally.
